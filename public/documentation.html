edieFlix API

Request URL Method
create new user http://localhost:8080/users POST
return all users http://localhost:8080/users GET
return one user by username http://localhost:8080/users/:Username GET
update user by username http://localhost:8080/users/:Username PUT
return all movies in API in a list http://localhost:8080/movies GET
Return data about a single movie http://localhost:8080/movies/:title GET
return data about a genre by name/title http://localhost:8080/genre/:Name GET
return data about a director by name http://localhost:8080/directors/:Name' GET
create a new user profile http://localhost:8080/users POST
add movie to favorites in user profile http://localhost:8080/users/:Username/Movies/:MovieID PUT
delete user profile http://localhost:8080/users/:Username DELETE
login http://localhost:8080/login POST
