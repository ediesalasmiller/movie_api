edieFlix API

Request URL Method
create new user /users POST returns JSON style array with Username,Password(hashed), favorite movies array, unique ID
return array containing all users created(auth token needed) /users GET
return one user by username(auth token needed) /users/:Username GET
update user by username (auth token needed) /users/:Username PUT
return all movies in API in a list /movies GET
Return data about a single movie /movies/:title GET
return description about a genre by name/title /genre/:Name GET
return data about a director by name /directors/:Name GET
add movie to favorites in user profile, return array of favorite movies in JSON format /users/:Username/Movies/:MovieID PUT
delete user profile /users/:Username DELETE
login and recieved authorization bearer token to access information within the API /login POST
