edieFlix API

Request URL Method
return all movies in API in a list http://localhost:8080/movies GET
Return data about a single movie http://localhost:8080/movies/title GET
return data about a genre by name/title http://localhost:8080/movies/genres/:title? GET
return data about a director by name http://localhost:8080/movies/directors/:name? GET
create a new user profile http://localhost:8080/account/newuser POST
update information for user profile http://localhost:8080/account/update PUT
add movie to favorites in user profile http://localhost:8080/account/forites/:title? POST
removie movie from favorites in user profile http://localhost:8080/account/favorites/:name? DELETE
update information for user profile http://localhost:8080/account/update PUT
update information for user profile http://localhost:8080/account/update/deactivate/:user? DELETE
